
########################################################################################
The SDK directories are the following :
- documentation 
	* functional-manual : useful functional documents for the Malima M2M services (pdf)  (advanced SDK only)
	* reference-manual : the help file for the SDK (phpdoc in HTML format)	
	* user-manual : the main guide to use the Malima M2M API (pdf)
- resources 
	* the properties files that you can use in your application (m2m.ini)
- sources  
	* the php code for the SDK. It contains a "sample" directory where you can get examples.
########################################################################################

If you are a new SDK user, please open the pdf user manual (in the documentation/user-manual directory). It will help you to start with the SDK.

Pre-requisite : please note that you need user credentials and a test sim card (provided by Orange via email) in order to start using the SDK.

N.B: the "advanced" SDK contains a "full" user manual, including the detailed description of all input/output parameters. 
It is not available on the internet. Please contact your Orange customer support to get the "advanced" version.
